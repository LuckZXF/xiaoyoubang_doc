##用户模块
###注册账号
####接口说明
判断是否存在该账号，没有则注册
####请求说明
请求方式：POST

请求地址：http://www.xiaoyoubang.applinzi.com/index.php/Home/Index/register
####请求参数
#####URL参数
|参数|说明|规定|
|---|---|---|
|user_account|注册账号|字母与数字搭配小于20位|
|user_name|注册者姓名|限10位汉字，字母或者数字|
|user_nickname|注册者昵称||
|user_password|注册密码|
|user_phone|注册手机号码|

####返回说明
返回：JSON数据

JSON数据格式

```
{
  "status": "0",
  "msg": "ok",
}
```

##帖子模块
##发帖子或者回复帖子
####接口说明
发送帖子 和 回复帖子同一接口 通过card_reply判断 0为发帖 1为回复(用途和金额和期限 前端分档后传0 1 2 3..)
####请求说明
请求方式：POST

请求地址：http://www.xiaoyoubang.applinzi.com/index.php/Home/Index/sendCard
####请求参数
####URL参数
>
{
    "user_account" : "eno122",
    "card_title" : "救命",
    "card_content" : "我要做手术啊",
    "card_reply" : "0",
    "card_borrow_used" : "0",
    "card_borrow_money" : "1",
    "card_borrow_time" : "2",
    "card_borrow_totalmoney" : "5000", //存真实金额
    "card_borrow_totaltime" : "2012-2-3"  //存还款时间 
}
>


####返回说明

json数据格式

```
{
  "status": "0",
  "msg": "ok"
}
```
